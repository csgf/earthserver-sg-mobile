About MERIS Repository

Repository of data coming from the MERIS spectrometer on board of the ENVISAT satellite has been created on the COMETA infrastructure with the gLibrary platform (https://glibrary.ct.infn.it).

This repository contains a subset of the MERIS Level3 demonstration products. More information on the dataset can be found here: http://earth.eo.esa.int/level3/meris-level3

gLibrary is a INFN platform to create and manage digital repository on e-infrastructure, providing easy-to-use front-ends and APIs that hides the underlying infrastrcture services.

The app implements a gLibrary client able to browse the repository and to download various kinds of digital assets and in various format (JPG and TIFF images, XML, HDF, NC data files) . The user can navigate the repository using a hierarchical filter mechanism based on asset metadata (product type, period, date) that allows to find easily the asset he is interested in. He can finally inspect the complete set of their metadata and proceed to the download from the infrastructure servers, finally viewing them on the device screen.
